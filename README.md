# Manipal-University-Clone-Website-Project
路This is an independent project developed using HTML,CSS,Javascript,PHP and SQL to develop a full stack website.

路This projects presents the website of the college Manipal University Jaipur with separate sections displaying various courses offered by the college,all the campuses,a student review section and much more.

路It also consists of a sign up/login page for teachers to create a record of students.

路All the login/sign up details are stored in the database and further student records created by the teacher are also stored in the database which can be further updated and deleted as well.
